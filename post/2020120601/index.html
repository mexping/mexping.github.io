Hugoのテーマ改変メモ

最終更新日 2020.12.06  /  投稿日 2020.12.05

Hugoのテーマ「Beautiful Hugo」を改変した時のメモ。

Archiveページの作成

config.tomlに下記を追加

 [taxonomies]
  tag = "tags"
  category = "categories"
  archive = "archives"

main.cssの .tags-headingがある場所に、 .archives-headingを追加

.intro-header .page-heading,
.intro-header .tags-heading,
.intro-header .categories-heading, 
.intro-header .archives-heading {
  text-align: center;
}

config.tomlに下記を追加。

[[menu.main]]
    name = "Archives"
    url = "archives"
    weight = 4

Chroma syntax highlighter

monokaiというスタイルを使いたかったので、CSSを生成して入れ替える。

hugo gen chromastyles --style monokai  > syntax.css

main.cssに下記を追記して、横スクロールできるようにする。!importantとしないと効かなかった。

.highlight pre code * { white-space: nowrap; // this sets all children inside to nowrap }
.highlight pre { overflow-x: auto !important; // this sets the scrolling in x }
.highlight pre code { white-space: pre; // forces <code> to respect <pre> formatting }

config.tomlのuseHLJSをfalseに変更。

[Params]
useHLJS = false

検索ページを作る

Googleカスタム検索でコードを取得し、config.tomlのgcseに貼り付け。カスタム検索のレイアウトは全幅にする。

[Params]
gcse = "#########"
